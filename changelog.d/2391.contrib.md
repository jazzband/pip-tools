The Sphinx configuration now resolves cross-references for
[`packaging`](https://packaging.pypa.io/en/stable/),
[`pip`](https://pip.pypa.io/en/stable/),
[`click`](https://click.palletsprojects.com/en/stable/#documentation),
{mod}`build`, and {mod}`importlib_metadata` via intersphinx, and the deprecated
`sphinx.util.console.bold` helper is replaced with a plain `logger.info` call
(the colour markup never rendered in CI logs anyway). Cross-references in
docstrings that previously fell back to nitpick suppression now link to the
upstream documentation -- by {user}`gaborbernat`.
